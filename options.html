How it works on claude.ai
1
Open claude.ai and start typing a prompt in the chat box
2
An ⚡ Optimize button appears next to the Send button
3
Click it — Prompt Forge reads your prompt and the current conversation for context
4
It fetches relevant example prompts from prompts.chat, then calls Groq to rewrite yours
5
The improved prompt is placed back in the chat box, ready to review and send
Popup (any site)
Click the Prompt Forge toolbar icon on any page to open the standalone popup. It searches prompts.chat directly and shows the best matching curated prompt — no API key required for this mode.
